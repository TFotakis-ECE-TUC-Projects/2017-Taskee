Κάθε task έχει ένα όνομα και έναν τύπο (πχ γυμναστική, διάβασμα (χωριζόμενο σε αποστήθηση και ασκήσεις), μάθημα, φαγητό, θέματα υγείας (πχ χάπια), κτλπ).
Επίσης, έχει ένα σύνολο από μέρη στα οποία μπορεί να γίνει το συγκεκριμένο task (πχ διαφορετικές αίθουσες μαθημάτων)
Κάθε task μπορεί να αλλάζει ή να μην αλλάζει μέσα στο πρόγραμμα - Tasks με δυνατότητα αλλαγής: ώρες του γυμναστηρίου, διαβάσματος - Tasks χωρίς την
 δυνατότητα αλλαγής: Μαθήματα, ραντεβού
Κάθε task που μπορεί να αλλάζει χαρακτηρίζεται από τη προτεραιότητά του σε σχέση με τα άλλα tasks (στην χειρότερη περίπτωση που ένα task δεν μπορεί να
 μπεί σε ένα δικό του slot, δηλαδή υπάρχει conflict με ένα task με ή χωρίς δυνατότητα αλλαγής, εμφανίζονται όλα τα tasks στο ίδιο slot ώστε ο χρήστης να
 επιλέξει manually την διαμόρφωση)

Κάθε task μπορεί να είναι επαναλαμβανόμενο η μη σύμφωνα με τους παρακάτω τρόπους:
1) Δεν επαναλαμβάνεται - Επίσκεψη στον γιατρό.
2) Επαναλαμβάνεται μέσα στην μέρα - Λήψη χαπιού πρωί βράδυ
3) Επαναλαμβάνεται κάθε έναν αριθμό από μέρες (πχ κάθε μέρα, μέρα παρά μέρα, κτλπ) - Λήψη ενός χαπιού, Πρόγραμμα γυμναστικής.
4) Επαναλαμβάνεται κάθε συγκεκριμένη μέρα/μέρες κάθε εβδομάδα (πχ κάθε Δευτέρα και Τετάρτη) - Μαθήματα
5) Επαναλαμβάνεται κάθε έναν αριθμό από μήνες (πχ κάθε μήνα, κάθε δύο μήνες, κτλπ) κατά τους παρακάτω τρόπους:
    α) Κάθε συγκεκριμένη ημερομηνία του μήνα (πχ κάθε 20 του μηνός) - Πληρωμές ενοικίων, Γυμναστηρίου, κτλπ
    β) Κάθε συγκεκριμένη μέρα του μήνα (πχ κάθε δεύτερη Τρίτη του μήνα)
6) Επαναλαμβάνεται κάθε χρόνο σε συγκεκριμένη ημερομηνία - Ετήσιο check-up υγείας
7) Επαναλαμβάνεται κατά έναν συγκεκριμένο αριθμό (πχ Δύο φορές την μέρα/εβδομάδα/μήνα/χρόνο) - Τρεις φορές την εβδομάδα γυμναστήριο
    (Δίνονται περιορισμοί από τον χρήστη για την συχνότητα του task δηλαδή πόσο κενό πρέπει να έχει η πρώτη επανάληψη του task από την δεύτερη)
8) Επαναλαμβάνεται με συγκεκριμένο αριθμό ωρών που πρέπει να κατανεμηθούν μέσα σε ένα χρονικό διάστημα (πχ μία βδομάδα/ένα μήνα/ένα χρόνο) - 70 ώρες
   διαδυκτιακών μαθημάτων μέσα σε 18 μέρες
    (Δίνονται από τον χρήστη περιορισμοί σχετικά με τον ελάχιστο και τον μέγιστο αριθμό ωρών καθώς και την συχνότητα του task δηλαδή πόσο κενό
    πρέπει να έχει η πρώτη επανάληψη του task από την δεύτερη)

Μπορούν να υπάρχουν ημερήσια προγράμματα τα οποία επαναλαμβάνονται κάθε έναν αριθμό από ημέρες (πχ κάθε μέρα, κάθε δύο μέρες)
Μπορούν να υπάρχουν εβδομαδιαία προγράμματα τα οποία επαναλαμβάνονται κάθε έναν αριθμό από εβδομάδες (πχ κάθε βδομάδα, κάθε δύο βδομάδες)
Μπορούν να υπάρχουν μηνιαία προγράμματα τα οποία επαναλαμβάνονται κάθε έναν αριθμό από μήνες (πχ κάθε μήνα, κάθε δυο μήνες)
Μπορούν να υπάρχουν ετήσια προγράμματα τα οποία επαναλαμβάνονται κάθε έναν αριθμό από έτη (πχ κάθε έτος, κάθε δυο έτη)

Τα ημερήσια προγράμματα αποτελούνται από tasks
Τα εβδομαδιαία προγράμματα αποτελούνται από tasks, ημερήσια προγράμματα
Τα μηνιαία προγράμματα αποτελούνται από tasks, ημερήσια προγράμματα, εβδομαδιαία προγράμματα
Τα ετήσια προγράμματα αποτελούνται από tasks, ημερήσια προγράμματα, εβδομαδιαία προγράμματα, μηνιαία προγράμματα
